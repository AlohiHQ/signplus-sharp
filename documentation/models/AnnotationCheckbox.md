# AnnotationCheckbox

Checkbox annotation (null if annotation is not a checkbox)

**Properties**

| Name      | Type                    | Required | Description                     |
| :-------- | :---------------------- | :------- | :------------------------------ |
| Checked\_ | bool                    | ❌       | Whether the checkbox is checked |
| Style     | AnnotationCheckboxStyle | ❌       | Style of the checkbox           |
